Personal Expense Tracker-Python

Overview
A simple and efficient Python-based application to track daily expenses, analyze spending habits, and gain insights into financial behavior..

Features
1.Add daily expenses (date, category, amount)
2.View all recorded expenses
3.Monthly expense summary
4.Category-wise expense breakdown
5.Pie chart visualization of spending
6.Identify highest spending category

Technologies Used
Python 3
Matplotlib (for visualization)
JSON (for data storage)

How to Run
-pip install matplotlib
python main.py
