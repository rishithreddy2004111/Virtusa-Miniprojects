**Online Retail Sales Analysis Database**



**Overview**

The Online Retail Sales Analysis Database is a SQL-based project designed to manage and analyze sales data of an online retail store. It demonstrates how relational databases and SQL queries can be used to extract meaningful business insights



**Features**

1.Top-Selling Products – Identifies most sold items

2.Most Valuable Customers – Finds highest spending customers

3.Monthly Revenue Analysis – Tracks revenue trends

4.Category-wise Sales – Compares performance of product categories

5.Inactive Customers – Detects customers with no orders



**Technologies Used**

MySQL Server

MySQL Workbench

SQL



**How to Run**

1.Install MySQL Server and MySQL Workbench

2.Open MySQL Workbench

3.Connect to local MySQL instance

4.Run SQL scripts in order:
