**Banking System Simulation-Java**



**Overview**

This project is a Java-based Banking System Simulation that demonstrates core banking operations using Object-Oriented Programming (OOP) concepts. It allows users to create accounts, perform transactions, and manage their banking activities in a simple console-based environment.



**Features**

1.Create account (Savings/Current)

2.Deposit, withdraw, transfer money

3.Transaction history

4.Basic authentication (username/password)



**Technologies Used**

Core Java (OOP: inheritance, encapsulation)

Collections framework



**How to Run**

**1.For Compile**

javac BankingSystem.java

**2.To Run**

java BankingSystem
